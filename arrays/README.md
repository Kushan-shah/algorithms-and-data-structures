# Arrays

This section contains solutions to common array-based problems, focusing on efficient approaches and standard problem-solving patterns.

---

## Concepts

* Traversal
* Two Pointers
* Sliding Window
* Prefix Sum
* Hashing (Map / Set)
* Sorting-based Approaches

---

## Problem List

* Two Sum — O(n), HashMap
* Best Time to Buy and Sell Stock — O(n)
* Contains Duplicate — O(n), HashSet
* Maximum Subarray (Kadane’s Algorithm) — O(n)
* Move Zeroes — Two Pointers
* Merge Sorted Arrays — Two Pointers
* Product of Array Except Self — Prefix/Suffix

---

## Approach

Problems are solved with emphasis on:

* Optimal time complexity
* Clean and readable code
* Standard patterns used in interviews

---

## Complexity

* Time: O(n) to O(n log n) depending on approach
* Space: O(1) to O(n)

---

## Notes

Arrays form the foundation for many advanced data structures and algorithms.
Strong command over array patterns is essential for solving higher-level problems efficiently.
