🔁 Backtracking

This section focuses on recursive exploration of all possible choices to build solutions.

---

📌 Overview

Backtracking is used when we need to generate all possible combinations or permutations by making decisions step-by-step and reverting them when needed.

---

🧠 Key Concepts

- Choose → Explore → Unchoose
- Recursion + state tracking
- Used array / path building
